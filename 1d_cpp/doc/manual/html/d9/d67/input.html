Overall Simulation Setup

Simulation Grid

  • NnodesX = ...
    The number of MPI processes.
  • numsp = ...
    Specifies the number of species. Just 1 for now.
  • l0 = ...
    The highest 'l' value to use in the velocity space spherical harmonic expansion.
  • m0 = ...
    The highest 'm' value to use in the velocity space spherical harmonic expansion.
  • nump = ...
    The number of cells that will make up the momentum-space axis in the simulation.
  • pmax = ...
    Specifies extent of cell centered momentum grid (0 < p < pmax).
  • mass = ...
    Species mass
  • charge = ...
    The extent of the absolute value of momentum to use in the simulation. pmin = pmax /(2.0* nump+1).
  • Nx = ...
    Number of cells along the X/Y directions.
  • xmin = ...
    Extent of simulation domain covered with Nx cells. Cell centered.
  • xmax = ...

Time Grid

  • clf_dp = ...
    Parameter that controls the minimum size of the time-step. The simulation step size (in simulation units) is _h_ = dt_out / float(integer(_dt_out_/clf_dp)+1). Please refer to the Clock object for more details.
  • n_outsteps = ...
    Number of outputs, evenly spaced from t=0 to t=t_stop.
  • n_distoutsteps = ...
    Number of output steps between distribution outputs.
  • t_stop = ...
    End simulation at this time (in simulation units).

Various Switches

Collision module

  • collisions_switch = [true | false]
  • f00_collisions = [implicit | explicit]
    Indicates which f00 collision push should be performed.
  • flm_collisions = [true | false]
    Indicates whether the flm collision push should be performed.
  • small_dt = ...
    Specifies the timestep for explicit f_00 collisions
  • smaller_dt = ...
    Specifies the timestep for laser heating loop using the explicit f00 collision formulation.
  • f00_exp_parabolic_approximation = ...
    Specifies the number of cells approximated using the parabolic approximation
  • f00_imp_RB_D_tolerance = ...
    Specifies the convergence tolerance for the D coefficient in the implicit f00 solver.
  • f00_imp_RB_D_iterations = ...
    Specifies the maximum number of iterations performed to calculate the Rosenbluth D coefficient in the implicit f00 solver.
  • flm_imp_tridiagonal = [true | false]
    Specifies whether full e-e collisions need to be performed for flm.

Vlasov Switches

  • implicit_B_push = [true | false]
    Specifies whether the implicit B field pusher should be used.

Field Solver

  • implicit_E = [true | false]
    Specifies whether implicit field solver should be used. Works great for collisional problems.

Initial Plasma Conditions

Density and Temperature

  • density_np = ...
    Density in units of 1/cm^3 that corresponds to a normalized simulation density of 1.0. Sets the real units for many of the quantities i.e. time
  • super_gaussian_distribution = ...
    Specify exponent of distribution according to SG theory. 2.0 specifies a Maxwellian.
  • pth_ref = ...
    reference thermal velocity. Only used for unit calculation at the beginning of simulation. Does not come into the code at all.
  • hydrocharge = ...
    Ion charge - used for electron-ion collisions

The plasma density and temperature can be initialized using the following. These quantities can be uniform in space, specified by a function, or by using a piecewise linear interpolation by using the following format (for a 1D system ranging from -100.0 to 100.0):

n(x) = cst 1.0
n(x) = fnc {1.0 + 0.01*sin(2*pi*x/200)}
n(x) = pcw {-100.0 1.0 100.0 1.0}

Note that the piecewise input needs to be supplied in pairs. The code requires
